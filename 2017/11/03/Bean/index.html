bean作用域：

  • singleton 每个springIOC容器一个bean定义一个对象实例
  • prototype 一个bean定义多个对象实例
  • request 一次Http请求一个bean定义一个实例，基于web的SpringApplicationContext有效
  • session 一个HTTPsession中，一个bean定义一个对象实例，基于web的SpringApplicationContext有效
  • global session 一个全局HTTPSession中，一个bean定义一个实例，基于web的SpringApplicationContext有效
