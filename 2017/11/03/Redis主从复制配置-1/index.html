Redis主从复制配置

  • redis
  • 2017-11-03

复制产生新的主服务器配置文件

redis.windows_M.conf
  1. 修改端口号为：6000
  2. 修改密码为：123456
    注：仅修改两项

复制产生新的从服务器配置文件

redis.windows_S1.conf
  1. 修改端口号为：6001
  2. 修改密码为：123456（建议与主服务器密码一样）

  3. 修改并设置主服务器地址与端口

  4. 修改连接主服务器的密码

用客户端连接–主服务器

redis-cli -p 6000

用客户端连接–从服务器

redis-cli -p 6001
